## 机房环境监测后台

此项目是一个简单的前后端分离项目,[前端](https://github.com/yujianyasuo/server_room_vue),[后端](https://github.com/yujianyasuo/server_room)

有以下功能

登陆,注册,验证码,表单显示,用户头像,图表
